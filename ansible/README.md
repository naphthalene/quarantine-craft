# Ansible Playbook

This is the playbook to deploy the server to a host over SSH.

## Required variables

These need to be configured by end user

* title

* web_fqdn
* web_certs_dir
* web_root_dir
* web_port

* srv_user
* srv_memory
* srv_tmux_session_name
* srv_backup_dir
* srv_overlay_dir
* srv_config_dir
* srv_data_dir
* srv_work_dir
* srv_plugin_jar_dir
* srv_server_jar_dir
